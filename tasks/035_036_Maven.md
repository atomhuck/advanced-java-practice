### Домашнее задание к видео: 035-036 Maven

**Тема:** Сборка проектов и управление зависимостями

**Задание:**
1. Создайте новый Maven проект вручную (через терминал или IDE).
2. Добавьте в `pom.xml` зависимость `Jackson Databind` для работы с JSON.
3. Напишите код, который сериализует объект Java в JSON строку и сохраняет в файл.
4. Добавьте плагин `maven-compiler-plugin` и укажите версию Java 17 (или ту, которую используете).

**Усложнение:**
Разделите проект на два модуля:
* `core` — содержит логику и модели.
* `app` — содержит `main` класс и зависит от `core`.
Соберите проект командой `mvn clean install` из корневой папки.
