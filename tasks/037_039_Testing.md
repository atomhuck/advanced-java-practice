### Домашнее задание к видео: 037-039 Тестирование (JUnit)

**Тема:** Юнит-тестирование

**Задание:**
Реализуйте класс `StringCalculator` с методом `int add(String numbers)`.
Метод принимает строку с числами, разделенными запятыми, и возвращает их сумму.
Пример: `"" -> 0`, `"1" -> 1`, `"1,2" -> 3`.

**Требования (TDD подход):**
1. Напишите тесты на базовые случаи.
2. Добавьте поддержку разделителя "новая строка" (`\n`) наравне с запятой.
3. Если передано отрицательное число — бросайте `IllegalArgumentException` с текстом "negatives not allowed: [список отрицательных чисел]". Напишите тест на это исключение.
4. Числа больше 1000 должны игнорироваться (например, `add("2,1001") -> 2`).

**Усложнение:**
Используйте `@ParameterizedTest` из JUnit 5, чтобы запустить один и тот же тест с разными наборами данных (строка ввода и ожидаемый результат).
