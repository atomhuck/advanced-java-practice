### Домашнее задание к видео: 012 Comparator и 013 Comparable

**Тема:** Сортировка объектов

**Задание:**
Создайте класс `Student` с полями: `name` (String), `age` (int), `score` (double), `group` (String).

**Задача:**
1. Сделайте `Student` реализующим `Comparable`. По умолчанию студенты сортируются по `score` (от большего к меньшему). Если баллы равны — по имени.
2. Создайте список из 20 случайных студентов. Отсортируйте его.
3. Создайте цепочку компараторов (используя `Comparator.comparing(...).thenComparing(...)`):
   * Сначала по группе.
   * Внутри группы по возрасту.
   * Внутри возраста по имени.

**Усложнение:**
Напишите универсальный компаратор `ReflectionComparator`, который принимает имя поля в конструкторе и сравнивает объекты по этому полю, используя Reflection API (это задел на будущее, но попробуйте загуглить).
