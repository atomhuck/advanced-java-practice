### Домашнее задание к видео: 040-041 Аннотации

**Тема:** Создание своих аннотаций

**Задание:**
1. Создайте аннотацию `@MethodInfo`, которая может применяться к методам и содержит поля: `author` (String, default "Unknown"), `date` (String), `revision` (int, default 1).
2. Пометьте несколько методов в своем классе этой аннотацией с разными параметрами.
3. Напишите класс-анализатор, который с помощью Reflection API проходит по всем методам вашего класса и выводит информацию из аннотаций в консоль.

**Усложнение:**
Создайте аннотацию `@Repeat(n)`. Напишите "раннер", который принимает объект и вызывает его методы, помеченные этой аннотацией, ровно `n` раз.
