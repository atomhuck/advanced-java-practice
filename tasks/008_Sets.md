### Домашнее задание к видео: 008 Множества Set

**Тема:** Множества (HashSet, TreeSet)

**Задание:**
Дан очень большой текстовый файл (можете сгенерировать его сами или взять любую книгу в .txt).

**Задача:**
1. Подсчитайте количество уникальных слов в файле.
2. Выведите топ-10 самых длинных слов, которые встречаются в тексте.
3. Выведите все слова, которые начинаются на букву 'A' в алфавитном порядке.

**Требования:**
* Используйте `HashSet` для быстрого поиска уникальных слов.
* Используйте `TreeSet` для получения отсортированного списка слов.
* Попробуйте реализовать "разность множеств": найдите слова, которые есть в тексте А, но которых нет в тексте Б (например, сравните две разные главы книги).

**Усложнение:**
Напишите свою реализацию `MyHashSet` на базе вашей `MyHashMap` из предыдущего задания.
