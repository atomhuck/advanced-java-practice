### Домашнее задание к видео: 009 Методы hashcode и equals и 010 Контракт

**Тема:** Контракт объектов в Java

**Задание:**
Создайте класс `ComplexKey`, который имеет поля: `int id`, `String name`, `long timestamp`, `double[] data`.

**Требования:**
1. Реализуйте `equals` и `hashCode` вручную (не генерируйте IDE).
2. Соблюдайте все правила: рефлексивность, симметричность, транзитивность, согласованность.
3. В `hashCode` используйте простое число (например, 31) для минимизации коллизий. Помните про `Arrays.hashCode()` для массива.

**Эксперимент (Важно!):**
1. Создайте `HashMap<ComplexKey, String>`.
2. Положите объект в карту.
3. Измените поле `name` у этого объекта (объект должен быть mutable для этого опыта).
4. Попробуйте достать объект из карты по этому же ключу. Объясните результат.
5. Вывод: почему ключи в HashMap должны быть **immutable**? Напишите ответ в комментариях к коду.
