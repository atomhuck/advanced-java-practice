### Домашнее задание к видео: 033-034 Лямбда-выражения и Stream API

**Тема:** Функциональное программирование в Java

**Задание:**
У вас есть список студентов `List<Student>`. У студента есть: имя, список оценок (`List<Integer>`), специальность, курс.

**Задача (используя ТОЛЬКО Stream API):**
1. Получите список всех уникальных специальностей студентов, чей средний балл > 4.5.
2. Сгруппируйте студентов по курсам.
3. Выведите для каждого курса средний балл всех студентов этого курса.
4. Найдите самого "успешного" студента (с максимальным средним баллом) на каждом курсе.
5. Преобразуйте список студентов в `Map`, где ключ — ID студента, а значение — его имя.

**Усложнение:**
Реализуйте свой собственный функциональный интерфейс `Condition<T>` с методом `boolean test(T t)` и напишите метод `filter(List<T> list, Condition<T> cond)`, который работает как стандартный фильтр, но принимает вашу лямбду.
