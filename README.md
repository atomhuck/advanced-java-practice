# Java Course Homework Project

Это проект с домашними заданиями по курсу "Продвинутая Java".

## Как пользоваться:
1. Открой эту папку (`JavaCourseHomework`) в IntelliJ IDEA как Maven проект.
2. Все описания задач находятся в папке `tasks/`.
3. Пиши свой код в соответствующих пакетах внутри `src/main/java/ru/course/`.
4. Тесты для модуля тестирования пиши в `src/test/java/ru/course/testing/`.

## Структура задач:
- **tasks/**: Описания всех домашних заданий в формате Markdown.
- **src/main/java/ru/course/**: Пакеты для реализации задач по темам (Collections, Multithreading и т.д.).
- **pom.xml**: Конфигурация проекта с уже добавленными JUnit 5 и Jackson для выполнения заданий.

Приятного обучения! Не забывай использовать Preview mode в IDEA для чтения задач.
